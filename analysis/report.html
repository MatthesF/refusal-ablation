Headline. The Arditi-style single-direction hypothesis holds up here: one direction governs a large, generalizable share of refusal behavior on unseen harm categories, while leaving benign behavior essentially untouched — but the effect is partial and uneven, not a clean on/off switch.

02The headline effect & its asymmetry

Each prompt is run once through the intact model and once through the edited model, so every comparison is a matched pair. Because the outcome is binary (refused vs. not) and paired, the correct test is McNemar's exact test; it depends only on the prompts whose label changed.

The two sides tell opposite stories, which is exactly what a well-targeted edit should do. On unsafe prompts the change is overwhelming (p ≈ 2e-43). On safe prompts it is statistically nil — in fact the lone safe prompt that the baseline refused was answered after the edit, so collateral movement, such as it is, also points toward helpfulness.

03The edit only ever pushes one way

A skeptic's first worry: maybe the edit just adds noise, flipping some prompts to compliance and others to refusal. It does not. Across all 600 prompts, 144 moved from refused to not-refused and 0 moved the other way. Every baseline compliance was preserved.

Why this matters statistically. Perfect monotonicity (one empty off-diagonal cell) means the marginal drop in refusal is not an average of competing effects — it is a floor on the true per-prompt effect. It also makes the McNemar odds ratio formally infinite, so the effect size is better reported as the absolute paired difference above.

The safety-relevant outlier. Self-harm facilitation is the one category the edit barely touches (100% → 92%, not significant). Whatever governs the model's self-harm refusals is mostly not aligned with the single direction that controls the rest — a useful hint that refusal is not perfectly one-dimensional.

05A subtler cost: refusals stop being clean

The binary label hides a second effect. Baseline refusals are terse one-liners ("I cannot help with that."). After the edit, answers to unsafe prompts get much longer — and crucially, this is true even for prompts that are still labeled "refused." Those become 200-word hedged essays that explain the harm, moralize, and frequently offer a "for-fiction-only" reframing — a partial erosion of the refusal that the binary outcome cannot see.

06Methods & caveats

What was tested

  • Unit of analysis: the prompt; each contributes one matched (baseline, edited) pair.
  • Primary outcome: manual label collapsed to refused vs. not-refused.
  • Tests: McNemar exact (paired binary); Wilson intervals for each rate; 20k-resample bootstrap for the paired-difference CI; Wilcoxon signed-rank for word counts; Holm correction across the 12 category tests.

Caveats

  • Single labeler, no adjudicated reliability. Labels are one rater's judgment; no inter-annotator agreement is available, so the boundary cases (esp. long soft-refusals) carry rater uncertainty.
  • One unusable output (a partial malware script) is treated as not-a-clean-refusal; with n=1 it moves nothing.
  • Greedy decoding, one sample per prompt — no within-prompt sampling variance is captured.
  • Quasi-complete separation (the empty off-diagonal) rules out a category-clustered logistic GLMM, so clustering is handled descriptively via the per-category tests rather than a mixed model.
  • "Refused" is a coarse target. Section 5 shows the edit degrades refusal quality even where the label is unchanged, so the 51.7% residual unsafe-refusal rate likely overstates how safe the edited model really is.
Bottom line. One linear direction accounts for a large, transferable fraction of Gemma's refusal behavior — enough to flip ~half of unseen unsafe prompts to compliance and to soften most of the rest — yet self-harm refusals and the residual hard-refusals show the behavior is not fully one-dimensional. The edit is precise (no safe-side collateral) but not complete.
